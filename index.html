Mis estudios

Para realizar el trabajo de crear una página web con CSS, sidebar, nav bar, etc., tuve que aprender sobre HTML, CSS, diseño web responsivo, sidebar, nav bar, frameworks como Bootstrap, entre otros.

HTML es el lenguaje de marcado que se utiliza para crear la estructura de la página web, mientras que CSS es el lenguaje de estilo que se utiliza para dar formato y diseño a la página web.

Además, aprendí a hacer que la página web sea responsiva y a utilizar herramientas como frameworks para acelerar el proceso de diseño y desarrollo. Con estos conocimientos, pude crear una página web con CSS, sidebar, nav bar, etc. de manera efectiva y eficiente.

Glosario de Sistemas Operativos

Archivos

Es la estructura general en la que se asignan nombres, se almacenan y organizan los archivos del sistema operativo.

Interfaz

Es el medio que permite la comunicación entre un usuario y una máquina, equipo, computadora o sistema operativo. En el contexto de los sistemas operativos, la interfaz es la forma en que el usuario interactúa con el sistema operativo, ya sea mediante una interfaz gráfica de usuario (GUI) o una línea de comandos.

Shell

Es la interfaz de línea de comandos que permite al usuario interactuar con el sistema operativo mediante la introducción de comandos.

Núcleo

Es la parte central del sistema operativo que se encarga de administrar los recursos del sistema y proporcionar servicios a los programas.

Multitarea

Es la capacidad del sistema operativo para ejecutar varias tareas al mismo tiempo.

Multiusuario

Es la capacidad del sistema operativo para permitir la ejecución simultánea de programas de diversos usuarios.

Memoria RAM

Es un dispositivo donde se guardan datos en forma temporal.

Procesador de texto

Es un programa que permite la manipulación de texto con formato y que permite generar archivos que conservan el estilo realizado.

Programa

Es un conjunto de instrucciones ordenadas correctamente que permiten realizar una tarea o trabajo específico.

Red

Es un conjunto de computadoras que están unidas entre sí, a través de elementos de comunicaciones, pueden ser permanentes o temporales, como enlaces telefónicos.

Software de sistema

Es el conjunto de programas que controlan los procesos básicos de una computadora y permiten el funcionamiento de otros programas.

Máquina virtual

Es un tipo de sistema operativo que presenta una interfaz a cada proceso, mostrando una máquina que aparece idéntica a la máquina real subyacente.

Página web

Es un documento dentro de un sitio de internet que contiene información.

Proceso

Es una operación o conjunto combinado de operaciones con datos, o bien una secuencia de acontecimientos definida, única y delimitada.

Glosario de Topologías de Red

Hub

Dispositivo que conecta múltiples dispositivos en una red de computadoras. Actúa como intermediario, recibiendo información de un dispositivo y reenviándola a los demás dispositivos conectados al Hub.

Switch

Dispositivo de red que conecta múltiples dispositivos en una red de computadoras. Al contrario que un Hub, el Switch solo envía información a los dispositivos que realmente la necesitan.

Router

Dispositivo de red que transmite datos entre dos o más redes.

Repetidor

Dispositivo de red que amplifica las señales de las redes locales, evitando así la pérdida de señal en áreas de baja potencia o distancia entre las fuentes de datos y los receptores.

DNS

Sistema que traduce los nombres de dominio en direcciones IP, permitiendo la navegación por internet de una forma más sencilla.

Access Point

Dispositivo que permite a los dispositivos móviles acceder a una red inalámbrica (Wi-Fi).

WAN

Red de área amplia (Wide Area Network), que se extiende por un área geográfica, como una ciudad, país o incluso continente.

LAN

Red de área local (Local Area Network), que se extiende por un área geográfica reducida, como una habitación o edificio.

MAN

Red de área metropolitana (Metropolitan Area Network), que se extiende por una zona urbana.

NIC

Tarjeta de interfaz de red, dispositivo que permite a una computadora conectarse a una red.

Protocolo

Conjunto de reglas que definen cómo se comunican las computadoras a través de una red.

Dirección IP

Dirección única asignada a cada dispositivo en una red, utilizada para identificar y ubicar a los dispositivos en una red de computadoras.

Gateway

Dispositivo o programa que permite la comunicación entre dos redes o la conexión a internet.

Firewall

Mecanismo de seguridad que protege a un sistema informático o a una red de accesos no autorizados.

Topología de red

Descripción de la forma en que los dispositivos de una red están conectados entre sí, es decir, cómo están interconectados y organizados.
